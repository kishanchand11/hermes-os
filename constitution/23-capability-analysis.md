# Part V – agency-agents

# 23. Capability Analysis

## 23.1 Purpose

Capability analysis determines required engineering abilities before implementation.

## 23.2 Process

Required capability → Available agents → Capability gaps → Validated solution

## 23.3 Requirements

Capability decisions must be based on verified requirements and evidence.

## 23.4 Gap Handling

Only verified gaps may require additional tooling or skills.
