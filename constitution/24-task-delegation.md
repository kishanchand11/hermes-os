# Part V – agency-agents

# 24. Task Delegation

## 24.1 Purpose

Task delegation defines how Hermes assigns engineering work to specialist agents.

## 24.2 Delegation Process

Hermes must:

1. Analyze the requirement.
2. Identify required capability.
3. Discover suitable agents.
4. Delegate work.
5. Validate results.

## 24.3 Delegation Rules

Tasks must only be assigned to verified capabilities.

## 24.4 Output Ownership

Hermes remains responsible for integration and final validation.
